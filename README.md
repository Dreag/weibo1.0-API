# weibo1.0-API
微博数据展示与舆情监控系统后台API

---- 

# 项目结构

## 架构说明

- 使用前后分离的REST架构风格构建系统
- 使用MySQL数据库进行持久化
- 使用Hadoop生态系统进行大数据的处理
- 使用HBase，MongoDB进行大数据的存储
- 使用TOMCAT8.x作为应用服务器

## 技术应用

- 后端使用框架Spring，SpringMVC,MyBatis进行开发
- 使用SpringBoot协助开发
- 使用Maven进行包管理

## 开发工具

- 使用GIT进行版本控制
- 使用IntelJ IDEA作为后台开发使用的IDE
- 使用POSTMAN进行后端的模拟请求

## 其他说明

- 后端API端口号8081
